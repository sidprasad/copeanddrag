CnD uses a combination of tools to compute and render visual layouts:

  • Cassowary checks whether the given set of constraints allows for a valid layout. If a solution exists, Cassowary will find it.

  • WebCola computes the final layout based on the satisfied constraints, optimizing the spatial arrangement of elements.

  • D3 helps implement visual directives such as color, icons, styling, etc.
